Universal Keys

  • ? - Show keyboard shortcut help

Function Selection Page

  • Enter - Select current function
  • Up/Down - Prev/Next function

Main ASM Page

  • Instruction Selection - New instruction must stay visible
  • Up/Down - Prev/Next instruction
  • Right Arrow -
    • On Jmp/Call - Go to target
    • On Ret - Go back to calling function
  • Left Arrow - Undoes previous Jump/Call
  • Enter - Open analysis window with source for current instruction

    • Analysis window open

  • Shift + Up/Down - Go up/down the function stack
  • Tab - Cycle through analysis tabs
  • Esc - close mini window at bottom (regardless of contents)
